# Superpowers for Gemini

Este archivo activa el framework de habilidades Superpowers. Los agentes de IA leerán este archivo para entender cómo utilizar las habilidades disponibles en este repositorio.

## Reglas de Oro

1. **Invocar habilidades antes de actuar**: Ante cualquier duda o inicio de tarea, consulta `skills/using-superpowers/SKILL.md`.
2. **Brainstorming obligatorio**: Antes de escribir código, se debe usar la habilidad de `brainstorming`.
3. **Planes de implementación**: No empieces a escribir código sin un plan detallado generado por `writing-plans`.
4. **TDD por defecto**: Se prefiere el desarrollo guiado por pruebas (`test-driven-development`).

## Habilidades Disponibles

- **Brainstorming**: Refinamiento de ideas y diseño previo.
- **Writing Plans**: Creación de planes de ejecución paso a paso.
- **Executing Plans**: Ejecución y seguimiento de tareas.
- **TDD**: Ciclo Red-Green-Refactor.
- **Systematic Debugging**: Proceso de depuración en 4 fases.
- **Code Review**: Listas de verificación de calidad.
- **Git Worktrees**: Gestión de ramas paralelas.
